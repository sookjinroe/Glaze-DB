# Glaze DB — 이중시유 시편 탐색기

중앙도재 유약 이중시유 시편 403종 + 단일 시유 26종 데이터베이스.

- `index.html` — 앱 본체 (정적, 빌드 불필요)
- `data.js` — 시편·유약 데이터 (specimens/glazes)
- `crops/` — 이중시유 시편 크롭 (파일명 = 코드)
- `singles/` — 단일 시유 크롭

코드 `24-14` = 24번 유약(옥색유) 2회 붓시유 후, 14번(상아색유) 2회 붓시유, 재벌.
보유·별점·태그는 localStorage(기기별 저장), 설정 탭에서 백업/이전 가능.
